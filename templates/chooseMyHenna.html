ChooseMyHenna

ChooseMyHenna is a web app designed to help users explore and select henna designs based on their personal preferences. The app allows users to specify design styles, then randomly generates Pinterest-sourced henna images to inspire their next look. Users can flip the image to their desired hand orientation, simplifying the process of finding the perfect designâ€”no more endless scrolling! It's built for simplicity and ease of use, with a clean interface that works seamlessly on both desktop and mobile devices.

Features

  • Design Preference Selection: Users choose between geometric or floral henna styles to match their aesthetic.
  • Hand Orientation Flip: Easily flip images to display on the left or right hand.
  • Random Design Generation: Provides random henna designs based on user preferences, sourced directly from Pinterest.
  • Mobile-Friendly Design: Optimized for both desktop and mobile for a smooth user experience.
  • Time-saving Functionality: Avoid endless scrolling by letting the app do the work of finding unique designs.
  • User-friendly Interface: A clean, intuitive interface with easy navigation.

Tech Stack

This project was built using: HTML, CSS, JavaScript, Python
